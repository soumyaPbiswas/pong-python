# SexyPong (Pygame)

A simple two-player Pong game built using Python and Pygame. Minimalistic, responsive, and a quick throwback to the arcade classic.

## Features
- Smooth gameplay at 60 FPS
- Real-time scoring
- Paddle collisions and wall rebounds
- Auto-reset after scoring

## Controls
- **Player 1**: W (up), S (down)
- **Player 2**: ↑ (up), ↓ (down)
- **Spacebar**: Start or continue the game
- **Esc**: Quit anytime
